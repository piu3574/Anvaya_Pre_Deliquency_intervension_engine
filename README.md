# Anvaya_Pre_Deliquency_intervension_engine

## Overview
Anvaya is a predictive analytics system designed to detect customers at risk of delinquency *before it happens*. It enables financial institutions to take proactive actions and minimize credit risk.

## Problem Statement
Traditional systems identify delinquency *after default occurs*, leading to financial losses and delayed interventions.

## Our Approach
We use a data-driven pipeline that includes:
- Feature engineering using **Weight of Evidence (WoE)**
- Feature selection using **Information Value (IV)**
- Machine learning models for risk prediction

## Features
- Early risk detection
- Interpretable scoring system
- Fast prediction pipeline
- Behavioral pattern analysis

## Tech Stack
- Python
- Flask (Backend)
- Scikit-learn
- Pandas, NumPy

## Workflow
1. Data Collection
2. Data Preprocessing
3. WoE Transformation
4. IV-based Feature Selection
5. Model Training
6. Risk Prediction
